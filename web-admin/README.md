## Running 

### Docker

Use the folowing commands to build and run the docker image

```
docker build -t web-admin .
docker run -p 8080:80 web-admin
```

### Local

```
npm install`
ng serve --port 8080`
```


[Click here to access the page](http://localhost:8080)
